2 The "Headless" Swap

// Built.at Dynamic Way - Powerful & Flexible

The Dynamic Code

<h1>${item[mapping.title]}</h1>
<audio src="${item[mapping.audio]}"></audio>

The Live Result

Whatever is in your Database

Song source: Dynamic URL from API

Why swap to Placeholders?

  • Zero Maintenance: You change the song file in the Grid, the developer's site updates automatically.
  • Schema Proof: If you rename your column from "Title" to "Label", the coder only changes one variable name.
  • Developer Freedom: They write the HTML/CSS they want. You provide the raw "fuel" (data).
